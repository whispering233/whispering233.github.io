    1. 首先需要循环遍历所有数字，逐个做判断，根据其定义自然有

      1. class PrimeNumber {
    public static void main(String[] args){
        // FLag
        boolean isFlag = true;
        for(int i = 2;i < 100;i++){
            for(int j = 2;j < 2;j++){
                if(i % j == 0){
                    isFlag = false;
                }
            }
            if(isFlag == true){
                System.Out.println(i);
            }
        //重置Flag
            isFlag = true
    }
}

Hexo 常用命令

安装

  1. npm install hexo -g
    1. 安装
  2. npm update hexo -g
    1. 升级
  3. hexo init
    1. 初始化

新建文章

  1. hexo new page <title> ; hexo n
  2. hexo new post <title>
    1. page 与 post 的区别
      1. post指文章、日志，会按时间顺序排列在博客上
      2. page指页面，不受时间规则影响，内容很少改变

发布

  1. hexo generate ; hexo g
    1. 生成静态页面至public目录
  2. hexo server
    1. 开启预览访问端口
  3. hexo deploy ; hexo d
    1. 部署到服务器
  4. hexo d -g
    1. 合并使用
