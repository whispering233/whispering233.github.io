LeetCode4

  1. 给定两个正序数组A、B，容量大小分别为m、n
  2. 找出返回此两个数组的中位数

题目分析

  1. 首先，两个数组已经有序

    1. 那么没有必要再次进行排序，即使是两个独立数组
    2. 只需要找到其中1个元素（奇数），或2个元素（偶数）

  2. 其次，中位数处于中间位置，左边部分 <= 中位数 <= 右边部分

    1. 则此问题类似于二分查找的变形
    2. 且中位数的位置已知
      1. (m+n) / 2
        1. 令k = (m+n) / 2
      2. ((m+n) / 2 )+1
    3. 此问题变为查找第k个元素

  3. 由于在两个独立数组中进行查找

    1. 问题在于如何在两个数组中分配 k 查找空间

    2. 假设均等分配且不越界

      1. 若A[k/2 - 1] < B[k/2 - 1]
      2. 则比A[k/2 - 1]小的数
        1. 最少有（k/2 - 1 ) 个
        2. 最多有（k-2）个
      3. 则A[K/2-1]最多是第k-1个数，而不是第k个数
      4. 则A[k/2-1]之前的数组元素都可以删去
      5. 重新赋值k

    3. 若m < (m+n) / 2

      1. 仅需比较最后一个元素
      2. 删去被去除的数组空间，重新赋值k

  1. 实现代码如下：

    1. public double findMedianSortedArraysHalf(int[] nums1, int[] nums2) {
        int length1 = nums1.length, length2 = nums2.length;
        int totalLength = length1 + length2;
        if (totalLength % 2 == 1) {
            int midIndex = totalLength / 2;
            return getKthElement(nums1, nums2, midIndex + 1);
        } else {
            int midIndex1 = totalLength / 2 - 1, midIndex2 = totalLength / 2;
            return (getKthElement(nums1, nums2, midIndex1 + 1) + getKthElement(nums1, nums2, midIndex2 + 1)) / 2.0;
        }
    }

    public int getKthElement(int[] nums1, int[] nums2, int k) {
        int length1 = nums1.length, length2 = nums2.length;
        int index1 = 0, index2 = 0;
        while (true) {
            // 边界情况
            if (index1 == length1) {
                return nums2[index2 + k - 1];
            }
            if (index2 == length2) {
                return nums1[index1 + k - 1];
            }
            if (k == 1) {
                return Math.min(nums1[index1], nums2[index2]);
            }
            int half = k / 2;
//            所以在这里进行了最小值判断
//            但在这里只是得到了新的索引值，而原数组的索引值缺失
//            所以如果元素数量偶数时，要调两次
            int newIndex1 = Math.min(index1 + half, length1) - 1;
            int newIndex2 = Math.min(index2 + half, length2) - 1;
//            如果k/2越界时，即length1远小于length2，此时k/2可能大于length1
//            此时第一次循环就要先选取较小长度数组的最后一个元素，若最后一个元素仍小于另一个数组的话，
//            则不能直接减去k/2个元素，而是要按照相应的元素数量进行去除
//            若最后一个元素较大，则正常进行循环，因为k值只在第一次给出，后续会不断更新
            int pivot1 = nums1[newIndex1], pivot2 = nums2[newIndex2];
            if (pivot1 <= pivot2) {
//                index现在还是0
//                减去绝对不是中位数的前k/2个数
//                现在只需要找到剩下的更新过的第k个数

                k -= (newIndex1 - index1 + 1);
//                指针移向下一个元素,看做新的数组再一次进行循环
//                当k=1时, 指向两个数组中较小一个
                index1 = newIndex1 + 1;
            } else {
                k -= (newIndex2 - index2 + 1);
                index2 = newIndex2 + 1;
            }
        }

//        作者：LeetCode-Solution
//        链接：https://leetcode-cn.com/problems/median-of-two-sorted-arrays/solution/xun-zhao-liang-ge-you-xu-shu-zu-de-zhong-wei-s-114/

    }

    2. 空间复杂度
      1. O(1)
    3. 其他
      1. 且当m+n是偶数时，需要调两次方法

LeetCode88

  1. 合并两个有序数组num1和nums2，将nums2合并到nums1中，使得nums1同样是有序数组

    1. 假设nums1的元素数量为m，nums2的元素数量为n

  2. 解题思路

    1. 创建新的数组，将排序好的数组赋值给num1，新数组的空间大小为(m+n)

      1. 已知两个数组有序，则类似于归并排序

        1. 关键在于其中一个数组的元素抽取完之后，将另一个没有被抽取完元素的数组的剩余值直接赋值给新数组

        2. 代码如下：

        3. class SolutionEightyEight {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        int[] nums3 = new int[m + n];
        int indexnums1 = 0;
        int indexnums2 = 0;
        for (int i = 0; i < m + n; i++) {
            if (indexnums1 == m) {
                nums3[i] = nums2[indexnums2++];
            } else if (indexnums2 == n) {
                nums3[i] = nums1[indexnums1++];
            } else if (indexnums1 < m || indexnums2 < n) {
                if (nums1[indexnums1] < nums2[indexnums2]) {
                    nums3[i] = nums1[indexnums1++];
                } else if (nums2[indexnums2] <= nums1[indexnums1]){
                    nums3[i] = nums2[indexnums2++];
                }
            }
        }
        for (int j = 0; j < m + n; j++) {
            nums1[j] = nums3[j];
        }
    }
}
