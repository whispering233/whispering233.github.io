Hexo 常用命令

安装

  1. npm install hexo -g
    1. 安装
  2. npm update hexo -g
    1. 升级
  3. hexo init
    1. 初始化

新建文章

  1. hexo new page <title> ; hexo n
  2. hexo new post <title>
    1. page 与 post 的区别
      1. post指文章、日志，会按时间顺序排列在博客上
      2. page指页面，不受时间规则影响，内容很少改变

发布

  1. hexo generate ; hexo g
    1. 生成静态页面至public目录
  2. hexo server
    1. 开启预览访问端口
  3. hexo deploy ; hexo d
    1. 部署到服务器
  4. hexo d -g
    1. 合并使用

其他

  1. hexo clean

    1. 清除缓存
    1. 只显示之前的内容
