求100以内所有质数

发表于 2021-04-17 分类于 算法 阅读次数： Valine：
本文字数： 335 阅读时长 ≈ 1 分钟

所有代码用Jave做示例

  1. 所谓质数，即是除1和其本身外不能被其他数整除

    1. 首先需要循环遍历所有数字，逐个做判断，根据其定义自然有

      1. class PrimeNumber {
    public static void main(String[] args){
        // FLag
        boolean isFlag = true;
        for(int i = 2;i < 100;i++){
            for(int j = 2;j < 2;j++){
                if(i % j == 0){
                    isFlag = false;
                }
            }
            if(isFlag == true){
                System.Out.println(i);
            }
        //重置Flag
            isFlag = true
    }
}
